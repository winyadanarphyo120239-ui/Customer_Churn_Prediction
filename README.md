# Customer_Churn_Prediction
A data science  project to predict customer churn using Python.

## Project Overview
This project uses machine learning models to predict customer churn based on historical data.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Model building using Python
- Accuracy evaluation and prediction

## How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run `python main.py`
